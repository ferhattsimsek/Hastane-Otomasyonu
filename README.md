# Hastane-Otomasyonu
 C# ve sql server kullanılarak kodlanmış hastane otomasyonu masaüstü uygulamasıdır, hasta-doktor ve sekreterden oluşan 3 panelden oluşur hastalar bölümü seçip doktorların müsaitliklerine göre randevu alabilir,doktorlar randevuları görebilir bilgilerini düzenleyebilir,sekreterler ise hem kendi hem doktorların hastaların biglilerini güncelleyebilir hemde duyuru yayımlayabilirler
